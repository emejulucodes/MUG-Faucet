# MUG Token - ERC20 Token with Faucet & Minting

A fully-featured ERC20 token contract built with OpenZeppelin and Foundry, featuring:

- Standard ERC20 functionality
- **requestToken()** - Faucet allowing users to claim 100 tokens every 24 hours
- **mint()** - Owner-only minting up to a MAX_SUPPLY of 10,000,000 tokens
